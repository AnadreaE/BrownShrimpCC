# BrownShrimpCC/R

Brown shrimp size-spectrum model  

## Description 

All functions that belong to the package are defined here (all those developed in the thesis work and also those implemented from other studies). These functions are organized in different files categorised as follows: 
- functions_SS_I: functions for the size spectrum model (Developed in stage I),
- functions_SS_II: functions for the size spectrum model (Developed in stage II),
- general: functions with other categories such as plots, read and adapt raw data from csv files, 
- growth_temming: functions developed by author Temming and used as a benchmark in this Thesis.

## Usage 

Only if any changes required this functions can be modified, e.g., for changing Parameter values for other species or different growth, development, reproduction, etc. functions. Otherwise the usage of all functions is documented in Roxygen2 Format and can be called in the console by running e.g. '?respiration_rate_b' (?function_name).
