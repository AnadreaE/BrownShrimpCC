# BrownShrimpCC/Simulations

Brown shrimp size-spectrum model  

## Description 

The two files found in this folder contains all simulations and plots that were includded in the Thesis related to this work. All figures from the Thesis can be reproduced in 'sims.R', whereas 'sims2.R' contains further tests and results that were not included directly in the Thesis, but considered for the better understanding of the results.
